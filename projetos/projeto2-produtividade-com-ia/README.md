# Projeto 2 – Produtividade com IA

# Objetivo
Demonstrar como Inteligência Artificial pode otimizar tarefas
de produtividade no dia a dia.

# Ferramenta usada
ChatGPT

# Como funciona
1. Definir tarefa ou rotina a ser automatizada
2. Configurar a IA para gerar soluções ou respostas
3. Avaliar resultado e ajustar para melhorar eficiência

# Benefícios
- Economiza tempo em tarefas repetitivas
- Garante organização e clareza
- Facilita estudo, trabalho e planejamento
